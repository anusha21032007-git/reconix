# Reconix - Vulnerability Scanner

Reconix is a CLI-based vulnerability scanner developed in Python for basic reconnaissance and security assessment.

## Features
- Port scanning
- Service banner detection
- Security findings summary
- Risk score calculation
- TXT report generation
- HTML report generation
- Multi-target scanning support

## Technologies
- Python
- Socket Programming
- File Handling

## How to Run

git clone <repo-link>
cd Reconix
python scanner.py
